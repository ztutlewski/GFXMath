Enumeration Type Documentation

enum gofxmath::FloatPrecision

Values that represent float precisions.

Remarks
Used as template arguments for various functions to determine approximation accuracies in floating point comparisons, and taylor approximations. Specifically used with FLOAT_PRECISION_VALUES and TRIG_PRECISION_VALUES.
Enumerator
HIGH 

The high precision option.

MEDIUM_HIGH 

The medium-high precision option.

MEDIUM 

The medium precision option.

MEDIUM_LOW 

The medium-low precision option.

LOW 

The low precision option.
