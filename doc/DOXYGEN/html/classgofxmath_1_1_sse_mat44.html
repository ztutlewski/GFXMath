Detailed Description

A 4x4 column-major matrix that uses four SseVecs for column vectors.

Returns
The SSE matrix 44.
Remarks
This class is used to take advantage of sse intrinsics for substantial performance gains in cpu-heavy rendering calculations.

Calculates the right-handed perspective projection matrix.

Date
2/21/2015
Parameters
nearThe distance to the near clip plane.
farThe distance to the far clip plane.
fovThe vertical field of view angle.
aspectThe aspect ratio (width / height).
Returns
The right-handed perspective projection matrix.
Remarks
This form of the perspective projection matrix takes on the following format:

\[ P_{m,n} = \begin{pmatrix} \cot(\theta) & 0 & 0 & 0 \\ 0 & \cot(\theta) * a & -f/(f-n) & -1 \\ 0 & 0 & -fn/(f-n) & 0 \end{pmatrix} \]

Where:

  • \(\theta\) is the fov (field of view) angle.
  • \(f\) is the distance to the far clipping plane.
  • \(n\) is the distance to the near clipping plane.
  • \(a\) is the aspect ratio of the viewport.

This is the SseMat44 equivalent of the Mat44 version of this function:
PerspectiveProjectionMatrix(float near, float far, float fov, float aspect, Mat44& result)

See also
void PerspectiveProjectionMatrix(float near, float far, float fov, float aspect, Mat44& result)
